## How to run

**Step 1:** Download and install Vagrant and VirtualBox

**Step 2:** Configure Virtual Machine

**Step 3:** Launch the Vagrant VM (vagrant up)

**Step 4:** cd /vagrant/catalog 

**Step 5:** python restaurantapp.py
